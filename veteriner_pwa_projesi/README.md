# Pati Dostu Veteriner - PWA Projesi

Bu proje, Web Tabanlı Mobil Uygulama Geliştirme dersi dönem sonu projesidir.

## Özellikler
- **PWA Desteği:** Kurulabilir (Manifest) ve Offline çalışabilir (Service Worker).
- **API Entegrasyonu:** catfact.ninja API kullanılarak rastgele kedi bilgileri ve ırk listesi çekilir.
- **CSS Çatısı:** Bootstrap 5 kullanılmıştır.
- **Yedek Mekanizma:** API çalışmadığında `data/backup.json` devreye girer.

## Kurulum
1. Dosyaları bir sunucuya yükleyin veya VS Code 'Live Server' eklentisi ile açın.
2. `images` klasörüne `icon-192.png` ve `icon-512.png` dosyalarını eklemeyi unutmayın.

## Kullanılan Teknolojiler
- HTML5, CSS3, JavaScript
- Bootstrap 5
- Fetch API & Service Workers
