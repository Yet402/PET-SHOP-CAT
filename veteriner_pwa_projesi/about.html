Proje Amacı

Bu proje, Web Tabanlı Mobil Uygulama Geliştirme dersi kapsamında, PWA teknolojileri kullanılarak geliştirilmiş bir Veteriner/Kedi Bilgi Sistemidir.

Teknik Detaylar

  • Kullanılan API: Catfact.ninja
  • CSS Çatısı: Bootstrap 5
  • Offline Çalışma: Service Worker ile HTML, CSS ve temel API verileri önbelleğe alınır.
Bu uygulama internet bağlantısı olmadığında da (Offline Mode) çalışacak şekilde tasarlanmıştır.
